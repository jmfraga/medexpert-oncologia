# Rúbrica de Evaluación — MedExpert Arena

## Instrucciones para el evaluador

Usted recibirá un conjunto de casos clínicos oncológicos. Para cada caso, se presentan **5 respuestas anonimizadas** (identificadas como A, B, C, D y E) generadas por diferentes sistemas de inteligencia artificial. Usted **no sabe** qué sistema generó cada respuesta.

Para cada respuesta, evalúe los **4 criterios** descritos a continuación usando una escala de **0 a 5 puntos**. Evalúe cada respuesta de forma independiente — no compare entre respuestas del mismo caso.

**Importante:**
- No investigue ni consulte guías durante la evaluación — use su criterio clínico experto
- Si una respuesta contiene información que usted considera incorrecta o peligrosa, refleje esto en la calificación y anótelo en los comentarios
- No hay respuestas "esperadas" — califique lo que lee, no lo que espera

---

## Escala general de puntuación

| Puntaje | Nivel | Descripción general |
|---------|-------|---------------------|
| **0** | Inaceptable | Completamente incorrecto, ausente o potencialmente dañino |
| **1** | Deficiente | Mayormente incorrecto, contiene errores graves que comprometen la atención del paciente |
| **2** | Insuficiente | Parcialmente correcto pero con errores significativos u omisiones importantes |
| **3** | Aceptable | Globalmente correcto con algunos errores menores o falta de detalle |
| **4** | Bueno | Correcto y completo con errores mínimos o de poca relevancia clínica |
| **5** | Excelente | Sin errores relevantes, información precisa y clínicamente sólida |

---

## Criterio 1 — Precisión diagnóstica (peso 25%)

> ¿El diagnóstico y la estadificación propuestos coinciden con lo esperado para el caso?

| Puntaje | Descriptor específico |
|---------|----------------------|
| **0** | Diagnóstico completamente erróneo o ausente |
| **1** | Diagnóstico incorrecto en tipo histológico o localización; estadificación equivocada |
| **2** | Diagnóstico parcialmente correcto (ej. tipo de cáncer correcto pero subtipo o estadio incorrectos) |
| **3** | Diagnóstico correcto, estadificación correcta pero sin mencionar biomarcadores o factores pronósticos relevantes |
| **4** | Diagnóstico y estadificación correctos, menciona biomarcadores principales, omite algún detalle menor |
| **5** | Diagnóstico preciso, estadificación completa (TNM/FIGO), biomarcadores correctos, factores pronósticos identificados |

### Aspectos a considerar:
- ¿El tipo histológico es correcto?
- ¿La estadificación (TNM, FIGO, Ann Arbor, etc.) es adecuada?
- ¿Se identifican correctamente los biomarcadores relevantes (HER2, EGFR, PD-L1, MSI, BRCA, etc.)?
- ¿Se mencionan factores pronósticos importantes para la toma de decisiones?

---

## Criterio 2 — Apego a guías clínicas (peso 30%)

> ¿Las recomendaciones son consistentes con guías clínicas vigentes (NCCN, ESMO, IMSS, etc.)? ¿Se citan correctamente?

| Puntaje | Descriptor específico |
|---------|----------------------|
| **0** | Recomendaciones contrarias a guías o potencialmente dañinas |
| **1** | Recomendaciones mayormente incorrectas; cita guías inexistentes o las aplica incorrectamente |
| **2** | Algunas recomendaciones son correctas pero hay errores significativos (ej. tratamiento de línea equivocada, fármaco contraindicado) |
| **3** | Recomendaciones generalmente correctas según guías, pero sin citar fuentes o con citas vagas |
| **4** | Recomendaciones correctas y alineadas con guías, cita fuentes aunque con imprecisiones menores (ej. año incorrecto) |
| **5** | Recomendaciones plenamente consistentes con guías vigentes, cita correctamente las fuentes (NCCN, ESMO, ensayos pivotales) |

### Aspectos a considerar:
- ¿El tratamiento propuesto corresponde a la línea correcta (1L, 2L)?
- ¿Los esquemas de quimioterapia/inmunoterapia son los recomendados por guías vigentes?
- ¿Se citan guías o ensayos clínicos relevantes?
- ¿Las citas son reales y están correctamente atribuidas?
- ¿Se evitan recomendaciones obsoletas o contraindicadas?

---

## Criterio 3 — Completitud (peso 25%)

> ¿La respuesta cubre todos los aspectos relevantes del manejo: tratamiento primario, alternativas, seguimiento, efectos adversos, consideraciones especiales?

| Puntaje | Descriptor específico |
|---------|----------------------|
| **0** | Respuesta extremadamente incompleta o solo aborda un aspecto aislado |
| **1** | Menciona solo el tratamiento primario sin alternativas, seguimiento ni consideraciones adicionales |
| **2** | Cubre tratamiento primario y algún otro aspecto, pero omite elementos importantes (ej. seguimiento, toxicidades) |
| **3** | Cubre tratamiento y alternativas, pero faltan aspectos como seguimiento, manejo de toxicidades o consideraciones especiales |
| **4** | Cubre la mayoría de aspectos relevantes; falta algún detalle menor o poco impacto clínico |
| **5** | Cobertura integral: tratamiento, alternativas, seguimiento, toxicidades, soporte, y consideraciones especiales cuando aplican |

### Aspectos a considerar:
- ¿Se menciona el tratamiento primario recomendado?
- ¿Se ofrecen alternativas terapéuticas?
- ¿Se describe el plan de seguimiento/vigilancia?
- ¿Se mencionan toxicidades esperadas y su manejo?
- ¿Se abordan consideraciones especiales (fertilidad, consejería genética, soporte nutricional, etc.)?
- ¿Se menciona la necesidad de estudios adicionales cuando aplica?

---

## Criterio 4 — Utilidad clínica (peso 20%)

> ¿La respuesta es práctica, accionable y útil para un oncólogo en consulta? ¿Ayudaría realmente en la toma de decisiones?

| Puntaje | Descriptor específico |
|---------|----------------------|
| **0** | Respuesta no tiene valor clínico; información irrelevante o incoherente |
| **1** | Respuesta muy genérica, no aporta más que conocimiento básico de libro de texto |
| **2** | Tiene algo de valor pero es demasiado vaga o teórica para ser útil en la práctica |
| **3** | Información correcta y parcialmente accionable, pero le falta especificidad (ej. no menciona dosis, esquemas concretos) |
| **4** | Práctica y accionable; incluye esquemas, dosis o parámetros concretos; algún detalle menor podría mejorar |
| **5** | Altamente accionable: especifica esquemas, dosis, secuencia temporal, criterios de decisión; podría usarse directamente como apoyo en consulta |

### Aspectos a considerar:
- ¿La respuesta está organizada de forma clara y lógica?
- ¿Incluye información específica (dosis, esquemas, intervalos)?
- ¿Un oncólogo podría tomar decisiones basándose en esta respuesta?
- ¿El lenguaje es profesional y adecuado para comunicación entre especialistas?
- ¿Distingue entre opciones preferidas y alternativas?
- ¿Prioriza la información de mayor relevancia clínica?

---

## Cálculo del puntaje compuesto

El puntaje compuesto ponderado se calcula como:

```
Compuesto = (Precisión diagnóstica × 0.25) + (Apego a guías × 0.30) + (Completitud × 0.25) + (Utilidad clínica × 0.20)
```

Rango: 0.0 – 5.0

| Rango compuesto | Interpretación |
|-----------------|----------------|
| 4.5 – 5.0 | Excelente — calidad de referencia |
| 3.5 – 4.4 | Bueno — clínicamente confiable con reservas menores |
| 2.5 – 3.4 | Aceptable — útil como apoyo pero requiere verificación |
| 1.5 – 2.4 | Insuficiente — no confiable para uso clínico sin supervisión estrecha |
| 0.0 – 1.4 | Inaceptable — potencialmente dañino |

## Comentarios libres (opcional pero valioso)

Para cada respuesta evaluada, se agradece cualquier anotación sobre:
- Errores clínicos específicos detectados
- Recomendaciones potencialmente peligrosas
- Aspectos particularmente bien logrados
- Cualquier observación que no se capture en los 4 criterios numéricos
